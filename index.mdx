  <Steps>
    <Step title="Choose an Algorithm">
      Select either the Middle Square Method (Cuadrados Medios) or Middle Product Method (Productos Medios) from the dropdown menu in the web interface.
    </Step>
    
    <Step title="Configure Parameters">
      Enter your seed value(s), the number of pseudorandom numbers to generate, and the digit precision (D parameter). For Middle Product, you'll need two seed values.
      
      ```javascript
      // Example configuration
      cantidadNumeros: 10
      semilla1: 1234
      d: 4  // 4-digit precision
      ```
    </Step>
    
    <Step title="Generate and Analyze">
      Click **Generar** to run the simulation. View the results table showing yi (squared/product value), xi (extracted middle digits), and ri (normalized pseudorandom number between 0 and 1).
    </Step>
  </Steps>
